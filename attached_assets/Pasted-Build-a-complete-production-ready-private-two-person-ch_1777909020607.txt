Build a complete, production-ready private two-person chat web app using Firebase. 
The app should feel modern, romantic, and intimate — designed exclusively for two people.

## Tech Stack
- Frontend: Vanilla HTML/CSS/JS (or React if preferred)
- Backend: Firebase (Firestore for messages, Firebase Auth for login, 
  Firebase Storage for image/file sharing)
- Hosting: Firebase Hosting ready (include firebase.json config)

## Core Features

### Authentication
- Email/password login (two fixed accounts: one for each person)
- "Remember me" option
- Auto-redirect to chat if already logged in
- Show partner's name and avatar at the top when logged in

### Chat
- Real-time messaging using Firestore onSnapshot listener
- Message bubbles: sent (right, accent color) vs received (left, neutral)
- Timestamps on each message (e.g., "Today 3:45 PM")
- Message status indicators: Sent ✓, Delivered ✓✓, Read ✓✓ (blue)
- Typing indicator ("She's typing..." with animated dots)
- Edit and delete your own messages (with "edited" label)

### Rich Messaging
- Emoji picker (use emoji-mart or a lightweight alternative)
- Image/photo sharing (upload to Firebase Storage, preview inline)
- Voice message recording (using MediaRecorder API, store in Storage)
- Reactions on messages (heart ❤️, laugh 😂, wow 😮 — long press to react)

### UX & Design
- Mobile-first, fully responsive layout
- Dark mode / light mode toggle with localStorage persistence
- Smooth animations for new messages (slide-in from bottom)
- Unread message count badge on browser tab (favicon badge)
- Sound notification for new messages (subtle chime, toggleable)
- Auto-scroll to latest message; "↓ New messages" button if scrolled up
- Infinite scroll / lazy load older messages (pagination with Firestore)

### Special/Romantic Touches
- "Last seen" status for partner
- A pinned section at the top for a shared "Note to each other" 
  (editable by both, auto-saved)
- A private shared photo album tab (grid view of all shared images)
- Anniversary / special date countdown widget in the sidebar
- Random "thinking of you" prompt button that generates a sweet 
  message starter

### Security
- Firestore Security Rules: only the two authenticated users can 
  read/write any data — no one else
- All messages stored only in their private Firestore collection
- No public endpoints; auth required for everything

## Deliverables
1. Full source code (HTML/CSS/JS or React components)
2. firestore.rules file with strict security rules
3. firebase.json and .firebaserc for deployment
4. A README with:
   - How to set up the Firebase project
   - How to add the two user accounts
   - How to deploy with `firebase deploy`

## Style Guide
- Font: Inter or Poppins
- Color palette: soft warm tones (rose, blush, cream) for light mode; 
  deep navy/charcoal for dark mode
- Rounded message bubbles, subtle shadow cards
- Micro-interactions: button press ripple, message send animation
- Overall vibe: cozy, intimate, premium — like a private sanctuary
